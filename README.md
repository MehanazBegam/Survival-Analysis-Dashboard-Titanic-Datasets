# Project Insights
-Female passengers had a significantly higher survival rate compared to males -1st Class passengers showed the highest survival probability -Survival chances decreased with lower passenger class -Higher fare values were associated with better survival outcomes -Most passengers embarked from Southampton (S) -Younger passengers showed relatively higher survival trends

# Final Conclusion
The analysis reveals that gender, passenger class, and fare were the most influential factors in passenger survival. The dashboard effectively converts complex data into clear, actionable insights, making it suitable for business intelligence reporting and data analytics portfolios.
